## About Rito DB

Rito DB is product store related software application.

## Software

PHP >= 7.2<br>
Apache/Nginx<br>
Mysql<br>
Composer<br>
Node Js<br>
Gulp<br>


## Installation

composer update<br>
npm install<br>
modify .env.example to .env<br>
php artisan migrate --seed<br>
php artisan serve<br>
gulp watch<br>
Run project in http://localhost:3000/<br>
